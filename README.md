ChiffonWebSite
==============

WebSite for a textile design collective, available in French & English.

Technology footprint
--------------------

* ASP.NET MVC 4
* I18N'ized
* Autofac
* MEF
* jQuery
* PowerShell
* MSBuild
* PSake

Requirements
------------

* .NET 4.5
* JRE v1.6
* PowerShell v3

Setup
-----

Add 'scripts\modules' to your PowerShell module path.
